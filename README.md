# AI Powered Real Estate Price Prediction Model

## 📌 Project Overview
This project is a Machine Learning based web application that predicts real estate prices based on user input features such as area, number of bedrooms, number of bathrooms, and property age.

The application provides a complete workflow that includes user registration, login authentication, a payment system using Stripe, and access to a machine learning prediction model.

The web interface is built using **Streamlit**, while the machine learning model is developed using **Scikit-learn**.

---

## 🚀 Features
- User Registration System
- Secure Login Authentication
- Payment Integration using Stripe
- Machine Learning Model for House Price Prediction
- Interactive Web Interface built with Streamlit

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/ritikmanocha/AI-Powered-Real-Estate-Price-Prediction-Model.git
```

### 2️⃣ Navigate to the Project Folder

```
cd AI-Powered-Real-Estate-Price-Prediction-Model
```

### 3️⃣ Install Required Libraries

```
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```
streamlit run main.py
```
